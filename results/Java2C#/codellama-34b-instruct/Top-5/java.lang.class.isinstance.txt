  Here are five C# APIs that have similar functionality to the Java API `java.lang.Class.isInstance()`:

1. `System.Object.ReferenceEquals(object, object)`: This method checks if two objects refer to the same instance.
2. `System.Type.IsAssignableFrom(Type)`: This method checks if a type is assignable from another type.
3. `System.Type.IsInstanceOfType(object)`: This method checks if an object is an instance of a particular type.
4. `System.Object.GetType()`: This method returns the type of an object.
5. `System.RuntimeTypeHandle.GetTypeByName(string)`: This method returns the type with the specified name.
