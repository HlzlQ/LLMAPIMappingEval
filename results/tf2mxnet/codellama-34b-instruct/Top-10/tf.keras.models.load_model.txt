  tf.keras.models.load_model() is not directly available in MXNet. However, there are some alternatives to load pre-trained models in MXNet:

1. mxnet.mod.Module.load(): This function loads a saved model from a file.
2. mxnet.mod.Module.load_checkpoint(): This function loads a saved checkpoint of a model from a file.
3. mxnet.mod.Module.load_params(): This function loads the parameters of a model from a file.
4. mxnet.mod.Module.load_optimizer(): This function loads the optimizer state of a model from a file.
5. mxnet.mod.Module.load_state_dict(): This function loads the state dictionary of a model from a file.
6. mxnet.mod.Module.load_parameters(): This function loads the parameters of a model from a file.
7. mxnet.mod.Module.load_optimizer_state_dict(): This function loads the optimizer state dictionary of a model from a file.
8. mxnet.mod.Module.load_model(): This function loads a saved model from a file.
9. mxnet.mod.Module.load_checkpoint(): This function loads a saved checkpoint of a model from a file.
10. mxnet.mod.Module.load_params(): This function loads the parameters of a model from a file.
