  tf.clip_by_value is a TensorFlow API for clipping the values of a tensor to a specified range. The equivalent MXNet API is:

* mxnet.ndarray.clip
* mxnet.ndarray.clip_by_value
* mxnet.ndarray.clip_by_percentile
* mxnet.ndarray.clip_by_norm
* mxnet.ndarray.clip_by_global_norm
* mxnet.ndarray.clip_by_std
* mxnet.ndarray.clip_by_mean
* mxnet.ndarray.clip_by_variance
* mxnet.ndarray.clip_by_skewness
